# ewbmcp

## There is no readme yet, figure it out :3

To install dependencies:

```bash
bun install
```

To run the server:

```bash
bun run server.ts
```

This will boot up an MCP server at `localhost:3000/api/mcp` or `/api/see` if using SEE.
You'll also need a Redis server for SEE, so spin one up locally :D

---

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
